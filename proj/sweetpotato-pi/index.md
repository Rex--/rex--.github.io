### Features:
 - Memory
    A. DRAM  - Datasheet says it only suports up to 512MB, however olimex and others use 1GB+ on their boards [DDR2/3]
    B. Flash - Maybe a couple GBs
    C. Card  - MicroSD Slot
 - Wifi via USB Module: MU7601
 - Sata via USB Bridge: GL830
 - USB x1 (OTG or Host?)
 - 5v Power Supply
