-----

### Pin Assignments

    Row 1 - PA1     |     Col 1 - PB2     |     LED - PA0
    Row 2 - PB1     |     Col 2 - PA4     |     RX  - PA6
    Row 3 - PB0     |     Col 3 - PA3     |     TX  - PA5

-----

### Resistor Values

R3 = 0ohm, R4 = NC
- If communicating with 5v device, use 0 ohm resistor on R1
- If communcating with 3.3v device use R1 = 1k, R2 = 2k
